## 🚀 Getting Started
### Prerequisites
* Node.js 18+
* GitHub account with admin access to test repositories
* Vercel account (free tier available)
### Installation steps
1. Clone the repository: `git clone https://github.com/jatin12-alt/Fixr.git`
2. Install dependencies: `npm install`
3. Set up environment variables: `cp .env.example .env`
4. Start the development server: `npm run dev`

## 🔑 Environment Variables
| Variable | Description | Required |
| --- | --- | --- |
| `DATABASE_URL` | PostgreSQL connection string | Yes |
| `VERCEL_TOKEN` | Vercel token for deployment | Yes |
| `GITHUB_TOKEN` | GitHub token for authentication | Yes |
| `CLERK_PUBLISHABLE_KEY` | Clerk publishable key for authentication | Yes |

## 🌐 Deployment
1. Set up a Vercel account and create a new project
2. Install the Vercel CLI: `npm install -g vercel`
3. Link your Vercel account to your GitHub repository
4. Configure your `vercel.json` file for deployment
5. Deploy your application: `vercel deploy`
